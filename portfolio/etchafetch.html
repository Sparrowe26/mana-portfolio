About the Game

Etch-A-Fetch is a small arcade web game where you must master the confusing controls of a classic Etch-A-Sketch to collect stars! There's also a doodle mode if you'd rather relax and draw on this digital Etch-A-Sketch.

  • Team size: Solo
  • Engine: PixiJS
  • Languages: HTML, CSS, JavaScript

Origin

Etch-A-Fetch is a game I created for a Web Development class I took in 2022. We were required to make a small web experience with PixiJS, so I decided to make a collectathon based around the Etch-A-Sketch.

My Contributions

This was a solo project, so I programmed the game as well as the website itself. All of the CSS styling and pixel art was made by me as well.
